# 图书管理系统 python756

一款基于 Python + Flask + MySQL 开发的图书馆管理系统，致力于提供高效便捷的图书管理解决方案。

## 技术栈

- 后端语言：Python
- 前端框架：Flask
- 数据库：MySQL, SQLite
- 开发工具：PyCharm
- 前端UI框架：Layui
- 数据库连接工具：Navicat

## 功能模块

### 系统角色

- 管理端
- 用户端

#### 管理端

- 用户登录和注册
- 图书信息管理（增删改查）

#### 用户端

- 图书信息查询
- 用户申请采购模块

## 运行环境

- 操作系统：Windows

## 部署步骤

1. 导入项目到 PyCharm
2. 安装依赖包
3. 配置数据库
4. 运行系统

## 目录结构

```plaintext
根目录/
├── app/              # Flask 应用目录
│   ├── templates/    # HTML 模板文件
│   └── static/       # 静态资源
├── venv/             # 虚拟环境
├── migrations/       # 数据库迁移文件
├── config.py         # 配置文件
└── run.py            # 程序入口
```
